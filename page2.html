Problem 2:Sum of Digits of a Number Problem Statement:Write an algorithm and pseudocode to find the sum of the digits of an integer. Example Input: Number = 1234 Expected Output: Sum = 10 (Use a while loop to extract digits)



ALGORITHM

STEP 1:START

STEP 2: DISPLAY ”ENTER THE NUMBER”

STEP 3: INPUT N

STEP 4:SUM ← 0

STEP 5: WHILE (N > 0) REM ← N%10 SUM ← SUM + REM N ← N/10

STEP 6:END WHILE

STEP 7:DISPLAY "SUM =", SUM

STEP 8: STOP

PSEUDOCODE

Begin

Use variables :N of type integer

Display ”Enter the Number”

Accept N

Sum=0

While (N>0)

{ Rem=N%10

Sum=Sum+Rem

N=N/10 }

Display”Sum=”,Sum End
